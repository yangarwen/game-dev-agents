Run the "story or world content" relay pipeline from the team charter (CLAUDE.md) for: **$ARGUMENTS**

Rules: ONE agent at a time. Pass each stage's conclusions forward. Stop and ask the user if tone or scope is undefined.

1. **narrative-designer** — Develop the content: beats, actual written text (logs, signage, dialogue), how it integrates with play. Within indie production limits.
2. **level-designer** — Placement: where each beat lands in the space, what the player is doing when they meet it, pacing fit.
3. **art-director** — Visual expression: what the beat needs visually (props, lighting, palette accents), within the established style and asset budget.

Final report: the story content with placement and visual notes, total production cost (words, props, setups), and a single combined "🌟 Tasks / Questions" list.
