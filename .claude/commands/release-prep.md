Run the "release preparation" relay pipeline from the team charter (CLAUDE.md). Target window (if given): **$ARGUMENTS**

Rules: ONE agent at a time. Pass each stage's conclusions forward. Stop and ask the user when a stage needs a decision only they can make.

1. **marketing-steam-strategist** — Assess launch readiness: positioning/hook strength, store page state (capsule, tags, copy, screenshots, trailer), wishlist trajectory, festival/press plan. Name what's missing.
2. **producer** — Slot the marketing gaps into the milestone plan: dates, owners, and what gets cut or moved to make the window real. Honest call on whether the window is achievable.
3. **legal-ip-advisor** — Pre-launch paperwork check: name clearance status, license manifest completeness, EULA/privacy policy needs, middleware attribution obligations.

Final report: a release-readiness scorecard (ready / at-risk / blocked per area), the dated action plan, and a single combined "🌟 Tasks / Questions" list.
